# Digital Twin

#### 介绍
本次是数字孪生首届工业大数据竞赛的算法的代码部分。
## 项目结构

方案一：基于多种机器学习方法的二次质检效率提升方案  
  
方案二：基于自编码器和LSTM方法的印刷不良品发生事件预测方案  

方案三：基于残差相似性模型的钢网印刷退化分析   


每一个方案文件下面都有如下对应的内容：  

1.  **数据预处理**  ： 对于数据集进行清洗，
2.  **数据分析**  ：数据分析
3.  **特征工程**  :
4.  **模型验证**   :
5.  **模型分析**    


**数据来源：**本次大赛官方给出数据
